* UPDATING THE OS VERSION

[[https://networkproguide.com/upgrade-cisco-ios-switch-step-by-step/][Upgrade Cisco IOS Switch Step-by-Step - NetworkProGuide]]

1. show current version of switch IOS version
HOSTNAME#show switch

2. download Latest cisco IOS image

3. Delete old IOS image if short on space
HOSTNAME# delete /f /r flash1:<SWITCH>.bin

~/f~ → force
~/r~ → recursive

Check that contents of flash are clear
<PC>#dir flash1:
<PC>#dir flash2:

4. Upload the New IOS to Flash
<PC># copy tftp flash1
>Address or name of remote host? 192.168.1.10 (SWITCH IP)
>source filename? c2960x-universalk9-mz.152-4.E6.bin
>Destination filename? c2960x-universalk9-mz.152-4.E6.bin
Verify that image is copied by checking flash again
<PC>#dir flash1:

Verify MD5 checksum for integrity
<PC># verify /md5 flash1:c2960x-universalk9-mz.152-4.E6.bin
