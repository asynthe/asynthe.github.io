_Roger notes_:

Start with the [Adam's] notes, take it seriously.
It's *hard*, not _technically hard_, but you need to know all the stuff.

- classroom-1 pinned notes
  find the latest set of Adam's CISSP bootcamp notes, D 1-8 (Domains)
- Mike Chappel's study guide.
- The CBK book

If you're a good reader, CBK book and Adam's notes are all you need.
_Don't read_ the NIST or ISO docs, there's *specific sections only*.
Adam states which in the notes.

The NIST docs are in Adams Zips, they're all free, easier to google than to find on the site.

If any of *Larry Gleenblatt's* courses come up, I'd recommend listening to him.
He has some old courses around the web you might be able to find, or you can buy a current. (I listened to them walking at night around the uni)

Adams stuff is at [[itprotv.com]], Larry is at [[itnetworkdefence.com]]
You can buy if you need a video, but _start with the books_.
