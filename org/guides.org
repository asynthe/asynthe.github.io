+ [[https://www.digitalocean.com/community/tutorials/how-to-manage-python-with-pyenv-and-direnv][How To Manage Python with Pyenv and Direnv - Digital Ocean]]
+ [[https://shivamarora.medium.com/a-guide-to-manage-your-environment-variables-in-a-better-way-using-direnv-2c1cd475c8e][A guide to manage your environment variables in a better way using direnv - Medium]]

** set up

1. Install direnv through your package manager
2. Hook into the shell

   in ~.bashrc~
   #+begin_src
   eval "$(direnv hook bash)"
   #+end_src

   in ~.zshrc~
   #+begin_src
   eval "$(direnv hook zsh)"
   #+end_src

It uses a ~.envrc~ file in the current and parent directories.
