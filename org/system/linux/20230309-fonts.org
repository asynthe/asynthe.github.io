* HOW TO SET UP - LINUX

*On a user level*
Copy and paste font into one of the next directories:
~~/.fonts~ or ~~/.local/share/fonts~

*On a system level*
Copy and paste into following directory:
~/usr/share/fonts~

Now, make the new fonts cache with
~$ fc-cache~ (all fonts directories) or
~$ fc-cache -vf /usr/share/fonts~ (specific directory)
~-v~ -> verbose
~-f~ -> force
