** manage audio with sndioctl

First, enable the audio server
~# sndioctl server.device=1~

Then, you can change the sound volume level in a absolute way
~# sndioctl output.level=0.8 # set it absolutely~

Or in a relative way
~# sndioctl output.level=-0.12 # decrease it relatively~
~# sndioctl output.level=+0.04 # increase it relatively~

Remember to use aliases or keybindings in KCSH[OpenBSD default shell]!
