** blur

[[https://askubuntu.com/questions/1002254/how-to-blur-an-image-from-command-line][How to blur an image from the command line? - StackExchange]]
[[https://imagemagick.org/Usage/blur/][ImageMagick Examples -- Blurring and Sharpening Images - imagemagick.org]]

Blur a file like this:
~$ convert <file>.jpg -blur 0x8 <output>.jpg~ 

Using Gaussian filter: (I prefer this method)
~$ convert <file>.jpg -filter Gaussian -blur 0x8 <output>.jpg~
