** keybindings


doom commands
```
doom sync
doom upgrade
doom doctor
doom env
```

doom command pallete commands
```
:q -> will quit and close emacs (will remember PATH because of the daemon)
:e PATH -> let's you edit a file
```

keybindings
```
M -> Meta (ALT)

M + x -> run any command available on emacs

Spc + . -> find-file
Space + f + r -> list recent files

Spc + b (buffer) + k (kill) -> kill open buffer

SPC + h + r + r -> restart Doom Emacs (remember to save)

VIM SPLITS
Ctrl-w + h -> horizontally
Ctrl-w + v -> split vertically
Ctrl-w + w -> change focus
Ctrl-w + c -> close

EMACS SPLIT
Space + w + s -> horizontal split
Space + w + v -> vertical split
Space + w + w -> change focus
Space + w + c -> close
```
