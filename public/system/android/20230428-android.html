-S / --turn-screen-off -> Turn screen off -w / --stay-awake -> Prevent device from sleeping (only USB connection)

adb (Android Debug Bridge)

NIXOS enable like this: 

{
  programs.adb.enable = true;
  users.users.<user>.extraGroups = ["adbusers"];
}

setup

  1. Start the server: $ adb start-server

  2. Now, enable Developer Mode on your phone:

    • Settings
    • About Phone
    • Software information
    • Build

    You can disable Developer Mode as follows:

    • Settings
    • System & Updates
    • Developer Options
    • Top blue slider, from On -> Off
  3. Enable USB Debugging:
    • Settings
    • System & Updates
    • Developer Options
    • DEBUGGING Section -> enable USB debugging
  4. Connect your PC and Android Phone with a USB cable and Allow the connection.
  5. Connect through ADB $ adb devices $ adb shell

note: Stack commands on your computer like this: $ adb shell pwd; ls /

conecting through wifi

note: Do it after setting a connection with a regular USB cable.

  1. We will have to open a TCP port first. Choose a number from 1000-19999, and open the port. $ adb tcpip 2233

  2. In your phone, go to

    • Settings
    • About phone
    • Status

    Note the IP address under 'IP address'. Use the IPv4, only numbers, no letters.

  3. On the terminal, connect to the IP address $ adb connect 10.0.0.191:2233

    To change to a USB Connection disconnect from all the devices: $ adb disconnect
