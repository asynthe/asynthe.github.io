Emacs keybindings (default)

movement

CTRL a -> move to start of the line CTRL e -> move to end of the line

CTRL f -> forward (one character) CTRL b -> backward (one character)

ALT f -> forward (one word) ALT b -> backward (one word)

undo / redo

CTRL u -> undo CTRL y -> redo

deleting

CTRL k -> delete everything after the cursor CTRL w -> delete last word ALT d -> delete word that is in front of the cursor (doesn't work for me)

inserting last command

ALT . -> insert an previously used argument

commands

CTRL p -> get the previous command in history (previous) CTRL n -> get the next command in history (next)

clear terminal

CTRL l -> clear terminal

vi keybindings

remember to press esc to go back to normal mode.

movement

0 (normal mode) -> go to start of line $ (normal mode) -> go to end of line

h -> move cursor back l -> move cursor forward

k -> goes a command back in history j -> goes a command forward in history

w -> forward (one word) e -> forward (last letter of word) b -> backward (one word) you can specify a number, so it goes <number> words back or forward example: 2b (two words backwards) 5w (five words forward)

undo / redo

u -> undo

deleting

you can add d to movement commands dw in this case will delete the word that is after the cursor

d0 -> delete entire line d -> cut full line

past commands

there's no way to do as emacs binding, but you can specify the last argument with !$

so if you only do !$ on a terminal, it will enter the last argument

example: 

$ echo "Hello"
Hello
$ prinf !$
prinf "Hello"

!! -> runs the last command again (helpful for long commands that need sudo/doas) sudo !!

command structure

you can run another command after the previous one is finished (successful or not) 

<command>; <command>

run the second command only if first command was successful 

<command> && <command>

helpful: let's say you want to do this script in one line 

#!/bin/bash
wget -q http://exemple.com/page1.php;
wget -q http://exemple.com/page2.php;
wget -q http://exemple.com/page3.php;

you can do it like this, with a for loop 

for P in {1..3} ; do wget -q http://exemple.com/page${P}.php ; done
