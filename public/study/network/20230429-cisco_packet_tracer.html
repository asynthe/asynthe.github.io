UBUNTU

How to install Cisco Packet Tracer on Ubuntu - SolveTechNow

run the next command on the Downloads folder. # dpkg -i <packet-tracer-file.deb

you can get a dialog problem, which you can solve with: # apt --fix-broken install then # apt install dialog

now run the first command again # dpkg -i <packet-tracer-file.deb

run it with $ packettracer
