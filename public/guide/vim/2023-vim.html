usage

keybindings

MODES i -> enter insert mode v -> enter visual mode V -> enter visual line mode

cw -> Change word (Deletes word on cursor then enters insert mode) o -> make a new line in insert mode

b -> previous word e -> next word

remember start of line with `start` key end of the line with `end` key

u -> undo (4u undo four last changes) Ctrl + R -> redo (or :redo:, or 4 + Ctrl-R)

gg -> jump to start of file G -> jump to end of file
