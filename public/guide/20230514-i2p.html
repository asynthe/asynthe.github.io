_prerequisite__ for running the main implementation of I2P, Java must be installed with

  • `dev-java/icedtea` or `dev-java/icedtea-bin`
  • `dev-java/openjdk` or `dev-java/openjdk-bin`
  • –oneshot `virtual/jdk`
  • –oneshot `virtual/jre`

OpenRC

_usage__ after enabling and starting the service, go on your preferred web broser and open ``` localhost:7657 in case of i2pd localhost:7070 ```

now we will make a Firefox profile to use with I2P

  • go to `about:profiles`
  • asd

click on `run this profile on a new browser`

on Firefox settings, go to `General` -> `Network Settings`

  • check `Manual proxy configuration`
  • change http proxy to

``` HTTP Proxy: 127.0.0.1 Port: 4444 ```

  • check for `SOCKS v5` to be enabled
  • add in `No proxy for`

``` localhost, 127.0.0.1 ```

Now, It happened to me that Firefox will try to run stuff you put on the url always like a search, you can disable this by

  • going to `about:config`
  • searcing for `keyw`
  • change `keyword.enable` from `true` to `false` (with that arrow button)
