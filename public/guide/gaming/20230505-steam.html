Steam will automatically download and set up Proton for your games, you have to set it up System-wide or Per-game, after which Steam will download the selected version.

Cog -> Properties -> Compatibility Tick the box of Force the use of a specific Steam Play compatibility tool And then choose the version of Proton you're running.

steam compatibility tool configuration failed

How to force uninstall or force update Proton Experimental w missing files - Steam Community "steam compatibility tool configuration failed" - Manjaro forum

I think my fix was something like this.

PROTON APPID's

APPID NAME
1493710 Proton Experimental
2348590 Proton 8.0
  • Go into SteamDB, and search for Proton.
  • Go into your custom steamapps folder or ~/.local/share/Steam/steamapps. (default)
  • Delete Proton appmanifest_<appid>.
  • Delete Proton folder in common.
  • Delete Proton folder in compatdata, if found.
  • Reboot Steam.
  • Install proton to another directory.
  • Go into that directory and copy Proton steamapps folder.
  • Copy that folder into the original steamapps.

Another thing that may have helped. Unchecking and rechecking the next settings.

  • Steam Shader Pre-Caching.
  • Allow background processing.
  • Enable Steam Play for supported titles.
  • Enable Steam Play for all other titles.

STEAM CMD

$ steamcmd cmd> login

If you want the steam-tui client you'll have to install it, but since it relies on `steamcmd` being set up, you'll have to set that up, just to generate the right files. nix-shell -p steamcmd –run steamcmd

And then log in: login <username> <password> <steam 2fa password> After that, steam-tui should start fine.

RUN A STEAM GAME FROM CLI

Is there a way to launch games via the terminal? - Steam community Command-Line Parameters - Steam Developer docs

find the steam game APPID in https://steamdb.info run it like this: $ steam steam://rungameid/<game-id>

try this: $ steam -applaunch <appID> [launch parameters]

you can then make a alias or a script, so you don't memorize the game id.

FIX: X Window Error

Make sure to:

  • Have Proton / ProtonGE updated (with protonup in case of ProtonGE)
  • Have your system packages updated, specially GPU drivers like nvidia-drivers

try: clogging the log with the same message, start like this $ steam > /dev/null 2>&1 & get rid of error and keep other steam messages $ steam 2>/dev/null
