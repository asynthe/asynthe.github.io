SIMPLE SSH FILE COPY

rsync

Transfer files from one place to another. You need rsync installed on both devices

$ rsync ~~/rsync matt@192.168.122.102:~/Downloads

rsync and Dropbox are about duplication of files and folders across multiples machines, with ability to do delta transfers, reason why they are used as backup.

scp

$ scp -r matt@192.168.122.102:~/test ~~/

  • syntax similar to rsync
  • best for individual files
  • simple, not many flags as rsync
  • best feature is transfer speed

SFTP

SFTP is an SSH subsystem, and is mean to replace scp. A good gui client for sftp and scp is FileZilla: easy transfer of multiple files and directories. Others are wormhole and croc

secure file transfer protocol $ sftp 192.168.122.102

  • very simple
  • access to the filesystem of the entire computer
  • allows access in a SSH manner, but also to send files to-from.
  • also allows you to navigate your host machine
  • a little more complicated than the others, because of permissions (user on main machine doesn't have permissions on remote server)
