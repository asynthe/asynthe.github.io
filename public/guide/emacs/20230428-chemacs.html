install and set up

note: you can put ~/.emacs-profiles.el in $XDG_CONFIG_HOME/chemacs/profile.el and ~/.emacs-profile in XDG_CONFIG_HOME/chemacs/profile.

from github page 

[ -f ~/.emacs ] && mv ~/.emacs ~/.emacs.bak
[ -d ~/.emacs.d ] && mv ~/.emacs.d ~/.emacs.default
git clone https://github.com/plexus/chemacs2.git ~/.emacs.d
  • this moves your emacs into ~/.emacs.default
  • and clones the directory into ~/.emacs.d

chemacs will only be "the emacs you boot from", this is, don't worry about ~/.emacs.d

there are two ways to run a custom emacs or emacs distribution from command line you can straight up run a emacs with custom configuration like this $ emacs --with-profile '((user-emacs-directory . "/path/to/config"))'

set up on file create a ~/.emacs-profiles.el file and add it this line 

(("default" . ((user-emacs-directory . "~/.emacs.default"))))

-> ~/.emacs.default will be the directory where emacs will load all the configs (init.el, config.el, etc…)

setting up another configuration

now, how to create another configuration, such as doom we will have to edit ~/.emacs-profiles.el like this 

(("default" . ((user-emacs-directory . "~/.emacs.default")))
 ("spacemacs" . ((user-emacs-directory . "~/spacemacs"))))

Doom Emacs

(("default" . ((user-emacs-directory . "~/.emacs.d")))
 ("doom" . ((user-emacs-directory . "~/doom-emacs")
           (env . (("DOOMDIR" . "~/doom-config"))))))

now, run doom emacs like this $ emacs --with-profile doom

or better yet, set it as the default when running emacs
