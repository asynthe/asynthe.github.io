package

usage

converting .wav to compressed .mp3

$ ffmpeg -i <input>.wav -vn -ar 44100 -ac 2 -b:a 192k <output>.mp3

flag description
-i the input (.wav file)
-vn disable video, no album cover
-ar audio sampling frequency
  8kHz (telcom), 44.1kHz (audio cd), 48kHz (movies)
-ac audio channels
  stereo (2), mono (1)
-b:a audio bitrate, convert to exact number
  in this example: 192kbit per second
   

media player (ffplay)

If built with sdl flag, a cli media player called ffplay is available

commands

ffplay <path/to/video> -> play a video curl https | xargs ffplay -> play a stream

more info: $ man 1 ffplay $ ffplay -h
