USAGE

$ gtklock -> lock screen $ gtklock -d -> lock screen and daemonize

Specify config file: $ gtklock -c /path/to/config.ini Specify style file: $ gtklock -s ./assets/example-style.css Load a module: $ gtklock -m /path/to/module.so Show options: $ gtklock –help-all

use gtklock when manually invoking the locker and gtklock -d when is invoked by something like swayidle.

CONFIGURATION

NIXOS

make sure to enable pam for gtklock in configuration.nix 

security.pam.services = {
  gtklock = { };
};
