keybindings

MAIN q -> quit (quit)

MOVEMENT k and <up> -> move up (up) j and <down> -> move down (down)

<c-u> and half-up (?) -> move half a page up (half-up) <c-d> and half-down (?) -> move half a page down (half-down)

<c-b> and <pgup> (Page Up) -> move a full page up (page-up) <c-f> and <pgdn> (Page Down) -> move a full page down (page-down)

h and <left> -> move to a parent directory (updir) l and <right> -> open a directory or open a file (open)

gg and <home> -> move to beginning of list (top) G and <end> -> mote to end of list (bottom)

SELECTION / CUT / PASTE <space> -> set selection in the current folder (toggle) v -> invert current selection (invert) u -> unmark all selections (unmark) y -> copy the current file or selection (copy) d -> cut " (cut) r -> rename file p -> paste " (paste) c -> clear copied or cut files (clear)

COMMAND INPUT : -> read builtin or custom command (read) $ -> run a command in shell (shell) % -> run a command in shell and see output in lf's ui (shell-pipe) ! -> run a command in shell and wait for key press (shell-wait) & -> run a command in the background (shell-async)

/ -> search a pattern ? -> search in opposite pattern direction (search-back) n -> find next file matching pattern (search-next) N -> find previous file matching pattern (search-prev)

KEYBINDINGS default keybindings z -> default keybindings provided s -> change the values of sortby and info options

e -> launch an editor i -> launch a pager w -> launch a shell
