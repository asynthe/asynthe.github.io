package

arch -> aria2 gentoo -> net-misc/aria2

usage

download torrents with $ aria2c '<magnet-link>' $ aria2c '<web-link-to-torrent-file> $ aria2c -T <path/to/torrent/file>

download multiple torrents like this: add the next to ~/aria2-torrents file 

torrent-link 1
torrent-link 2
torrent-link 3

start the download with $ aria2c -i ~~/aria2-torrents

remember to delete or clear the file after $ rm ~/aria2-torrents or $ echo ' ' > ~/aria2-torrents
