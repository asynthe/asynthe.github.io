GAMEMODE

"daemon/lib combo for Linux that allows games to request a set of optimisations be temporarily applied to the host OS and/or a game process."

github page arch wiki

PACKAGE arch -> ... gentoo -> games-util/gamemode nixos -> gamemode

USAGE launch the game like this: $ gamemoderun ./game or in Steam launch options $ gamemoderun %command%

check if gamemode is running (when game is open) $ gamemoded -s

check if gamemode works $ gamemoded -t

CONFIGURATION configuration is made in gamemode.ini, which can be located in:

  1. /etc/
  2. /usr/share/gamemode/

you can also set in $PWD and $XDG_CONFIG_HOME / $HOME/.config/, but GPU settings won't have effect here.

note: add to steam as gamemoderun %command% or to lutris
