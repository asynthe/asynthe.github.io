package

arch -> gentoo -> nixos -> sshfs

usage

There are two ways to establish a connection to a remote machine

  1. Using password authentication
  2. Using a SSH key pair (recommended)

# sshfs [-o <options>] <remote user>@<remote host>:/<path to remote directory> /mnt/<folder name>/

# sshfs -o allow_other,IdentityFile=/home/kb/.ssh/id_rsa ubuntu@131.153.142.254:/home/user/ /home/asynthe/debian-pc
