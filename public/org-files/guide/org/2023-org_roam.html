Main keybinds

SPC n r n -> org roam capture SPC n r i -> insert a node link SPC n r r -> see backlinks

Nodes

a node is a simply note or a heading with an ID property, which will make it identifiable to org-roam

Creating a node

you make regular org-roam nodes with org-roam-capture (SPC n r n)

you can also put a node on a regular heading with: org-id-get-create (SPC m I)

Aliases for nodes

easily create an alias while in a node :PROPERTIES: with: org-roam-alias-add (SPC m m o a)

Completion at point

using completion-at-point you can auto-complete words with their respective org-roam node link.

add this to your ~/.doom.d/config.el 

(map! :leader
      :desc "Completion at point"
      "n r c" #'completion-at-point)

Database Sync

convert a file for org roam with ~org-id-create

in case of deleting, renaming, or moving notes database.

  • clean the database with org-roam-db-clear-all
  • re-scan the database with org-roam-db-sync or M-SPC n r s

Dailies

today capture a note for today org-roam-dailies-capture-today with SPC n r d T go to today's notes org-roam-dailies-goto-today with SPC n r d t

specific date capture a note for a specific date org-roam-capture-date with SPC n r d D go to a specific date note org-roam-goto-date with SPC n r d d (note: move around dates with SHIFT + ARROW KEYS, can also use mouse)

tomorrow capture a note for tomorrow org-roam-dailies-capture-tomorrow with SPC n r d M go to tomorrow's notes org-roam-dailies-goto-today with SPC n r d m

yesterday capture a note for yesterday org-roam-dailies-capture-yesterday with SPC n r d Y go to yesterday's notes org-roam-dailies-goto-yesterday with SPC n r d y
