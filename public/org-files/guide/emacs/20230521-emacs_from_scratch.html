Emacs from Scratch

useful when setting up emacs

  • start install by installing emacs
  • install chemacs
  • after getting emacs –with-profile default, begin literate config
  • install super save

installation

Ctrl-x Ctrl-f -> find-file then edit ~/.emacs.d/init.el

to zoom in use the command with M-X it should be text-scale-increase

paste into the file 

(org-babel-load-file
 (expand-file-name
  "config.org"
  user-emacs-directory))

note: be careful with writing elisp instead of emacs-lisp as this will not work.

now on the same directory, create ~/.emacs.d/config.org document (remember find-file)

initial config.org

Now save with C-x C-s, close and open again Emacs note: you can also do M-x load-file command, then write ~/.emacs.d/init.el

overview of the set up

versions to run with [chemacs] ~/.emacs.d -> chemacs ~/.emacs main emacs with literate config ~/.doomemacs -> doom emacs

using emacs daemon

-> try to always use the emacs daemon, as it will be faster and always available.

run it with $ emacs --daemon & (will run the emacs profile in ~/.config/chemacs/profile)

in case of running with a certain profile $ emacs --with-profile doom --daemon &

now to open emacs instances, use $ emacsclient -c -s <profile> -a emacs for doom (if doom profile) $ emacsclient -c -s doom -a emacs

note: add this in your twm configuration or .Xprofile 2nd note: change the desktop file of emacs to run the emacsclient command ($ emacsclient -c -s emacs -a emacs)
