keybinds

MOVEMENT Space / Page Down -> move down the screen Shift + Space / Page Up -> move up the screen Ctrl + up arrow / Home -> top of page Ctrl + down arrow / End -> bottom of page

Tab -> focus on next element, move around the page Shift + Tab -> focus on previous element, move back

Alt + left arrow / Ctrl + [ -> Previous page Alt + right arrow / Ctrl + ] -> Next page

EDITING Ctrl + C -> copy Ctrl + X -> cut Ctrl + V -> paste Ctrl + Shift + V -> paste (plain text) Ctrl + Z -> undo Ctrl + Y / Ctrl + Shift + Z -> redo Ctrl + A -> select all Ctrl + P -> print

Ctrl + left arrow -> go one word to the left Ctrl + right arrow -> go one word to the right Home -> go to beggining of line End -> go to end of line Ctrl + Home -> beggining of text Ctrl + End -> end of text

Delete -> delete character Ctrl + Backspace -> delete word to the left Ctrl + Delete -> delete word to the right

Ctrl + + -> zoom in Ctrl + - -> zoom out Ctrl + 0 -> zoom reset

Ctrl + M -> mute / unmute audio Ctrl + F -> search on page / -> quick search Ctrl + K / Ctrl + J -> search on search bar (+ Alt + up arrow / Alt + down arrow -> change search engine)

Ctrl + Q -> quit firefox Ctrl + R / F5 -> Reload Ctrl + F5 / Ctrl + Shift + R -> Reload (override cache) Esc -> Stop F11 -> toggle full screen

TABS Ctrl + T -> open a new tab Ctrl + W -> close a tab Ctrl + Shift + T -> open previously closed tab

Alt + 1 to 8 -> go to tab from 1 to 8 Alt + 9 -> go to last tab Ctrl + Tab / Ctrl + Page Down -> go one tab to the right Ctrl + Shift + Tab / Ctrl + Page Up -> go one tab to the left

Ctrl + Shift + Page Up -> move tab left Ctrl + Shift + Page Down -> move tab right

WINDOWS Ctrl + N -> open a new window Ctrl + Shift + P -> open a private window Ctrl + Shift + W / Alt + F4 -> close window

TOOLS Ctrl + Shift + Y -> downloads Ctrl + Shift + A -> add-ons Ctrl + Shift + I / F12 -> developer tools Ctrl + Shift + K -> web console Ctrl + Shift + C -> inspector Ctrl + Shift + S -> screenshot Ctrl + Shift + E -> page source Ctrl + I -> page info

HISTORY Ctrl + H -> history sidebar Ctrl + Shift + H -> history on another window Ctrl + Shift + Del -> clear recent history

PDF VIEWER N / J or right arrow -> next page P / K or left arrow -> previous page Ctrl + + -> zoom in Ctrl + - -> zoom out Ctrl + 0 -> automatic zoom R / Shift + R -> rotate / counter-rotate Ctrl + Alt + P -> presentation mode S -> selection tool H -> hand tool
