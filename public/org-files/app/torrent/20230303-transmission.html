set-up

initialize the daemon

systemd $ systemctl enable transmission.service $ systemctl start transmission.service

OpenRC $ rc-service

update your download directory $ transmission-daemon --download-dir "~~/Downloads/"

check your current settings with $ transmission-daemon --dump-settings

check more settings with $ man transmission-daemon

start the daemon $ transmission-daemon

using transmission add a torrent, you can use a downloaded torrent path or direct URL or even a magnet link $ transmission-remote -a "/home/meow/x/" $ transmission-remote -a "magnet:?xt=urn:..."

download and check the status of your downloads with the list command $ transmission-remote -l

remove a torrent, use the ID number (left-hand column on the list command) to select which torrent you want to remove $ transmission-remote -t (torrentid-1),(torrentid-2) -r

you can also remove everything $ transmission-remote -t all -r

stop transmission $ transmission-remote --exit

usage

add a torrent like this $ transmission-remote -a <path/to/.torrent> $ transmission-remote -a "<magnet-link>"

see the progress with $ transmission-remote -l

pause a torrent $ transmission-remote -t <torrent-number>, <torrent-number-2> -S pause all the torrents $ transmission-remote -t all --stop

start it again $ transmission-remote -t <torrent-number> --start

remove a torrent $ transmission-remote -t <torrent-number> -r

asd

see the contents of a torrent $ trasmission-remote -t <torrent-number> -f

if you want to see live progress of your torrents, you can enter the WebUI search localhost:9091 in your preferred browser

or use watch $ watch transmission-remote -l

tmux

i always use watch, which you can come with your linux distro or you can obtain through busybox or package manager and use it inside tmux as $ watch transmission-remote -l
