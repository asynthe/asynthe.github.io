keybinds

NAVIGATION . -> next frame , -> previous frame left arrow -> -5 seconds right arrow -> +5 seconds up arrow -> +60 seconds down arrow -> -60 seconds Page Up / Down -> next / previous chapter

Shift + left -> -1 second (no osc) Shift + right arrow -> +1 second (no osc) Shift + up arrow -> +5 seconds Shift + down arrow -> -5 seconds Shift + Page Up / Down -> move 600 seconds

PLAYLIST Shift + < -> previous video (in playlist) Shift + > -> next video (in playlist) < -> next playlist link > -> next playlist link F8 -> see playlist items

MENU p -> play / pause q / Ctrl + c -> quit, save current position f -> fullscreen mode m -> mute v -> subs on / off o -> show progress s -> screenshot

Shift + o -> toggle show progress Shift + s -> screenshot w/o subtitles Shift + t -> toggle stay on top

Ctrl + h -> cycle hardware decoding Shift + a -> Cycle aspect radio ("16:9","4:3"…)

PLAYBACK Ctrl + + -> increase audio delay Ctrl + - -> decrease audio delay [ -> decrease speed ] -> increase speed 1 -> decrease contrast 2 -> increase contrast 3 -> decrease brightness 4 -> increase brightness 5 -> decrease gamma 6 -> increase gamma 7 -> decrease saturation 8 -> increase saturation 9 -> decrease audio volume 0 -> increase audio volume w -> zoom out e -> zoom in

SUBS j -> next sub Shift + j -> previous sub z -> increase sub delay x -> decreate sub delay r -> move sub up t -> move sub down

Ctrl + left arrow -> skip to previous sub Ctrl + right arrow -> skip to next sub

Ctrl + Shift + left arrow -> adjust timing to previous sub Ctrl + Shift + right arrow -> adjust timing to next sub

VIDEO RECTANGLE Alt + left arrow -> video pan-x 0.1 Alt + right arrow -> video pan-x -0.1 Alt + up arrow -> video pan-y 0.1 Alt + down arrow -> video pan-y -0.1 Alt + Backspace -> reset video zoom and pan

usage

creating a videos playlist

create a my_list.txt file with the list of files

example: 

video1.mp4
video2.mp4
video3.mp4

then run: $ mpv --playlist=my_list.txt

while this works, you may also change .txt for .m3u for more compatibility with apps (like mpvpaper)

mpvpaper

"Videos as a wallpaper (Wayland only)."

github page

package

arch -> ... gentoo -> gui-app/mpvpaper through guru overlay

usage

run video in loop and fork to background as this: $ mpvpaper -f -o "loop-file=inf" <display> <file>

now, if you're running resource-intensive processes like playing a videogame, you would want your background video to pause or to stop, in this case add the -p flag for auto-pause, or -s for auto-stop. $ mpvpaper -f -o "loop-file=inf" <display> <file>

PLAYLISTS be sure to have your playlist file as .m3u

FLAGS (from man page) -h, --help -> display options -v, --verbose -> be verbose -f, --fork -> forks mpvpaper -p, --auto-pause -> automatically pause mpv when hidden -s, --auto-stop -> automatically stop mpv when hidden -o, --mpv-options -> options to run with mpv, they have to be inside a quote "like this".
