keybindings

h / left arrow -> move left j / down arrow -> move down k / up arrow -> move up l / right arrow -> move right

shift + h -> move to start of page shift + j / Space -> next page shift + k / Shift + Space -> previous page shift + l -> move to end of page

gg -> go to first page G -> go to last page nG -> go to n page

= -> reset to normal size - -> zoom out shift + = (+) -> zoom in

TAB -> show TOC (table of contents) q -> quit a -> fit page s -> fit width f -> follow link d -> dual-page view r -> rotate 90 defrees R -> reload document ctrl + r -> invert color ctrl + n -> toggle status bar F5 -> presentation mode F11 -> fullscreen mode

configuration

Zathura configuration is done inside ~/.config/zathura/zathurarc

Copy to clipboard enable in zathurarc like this set selection-clipboard clipboard

Make zathura the default pdf viewer $ xdg-mime default org.pwmt.zathura.desktop application/pdf

Read Microsoft Office / LibreOffice using zaread-git (aur) $ zaread /path/to/document.docx

Sandbox mode secure environment using a strict seccomp filer. set in zathurarc like this set sandbox strict
