kittens

icat (images)

doc page

See images in your terminal with: $ kitty +kitten icat <image>.jpeg or $ kitten icat <image>.jpeg

you can make an alias alias icat="kitty +kitten icat"

note: Install ImageMagick for more image formats than the supported PNG/JPG/GIF/BMP/TIFF/WEBP note: It works in SSH.

FLAGS --align <ALIGN> -> Horizontal alignment of the image, values are: center, left, right.
