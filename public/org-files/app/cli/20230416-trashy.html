package

cargo -> trashy arch -> trashy (aur) gentoo -> ... nixos -> trashy

usage

Put into trash bin $ trash <path1> <path2> <path3> or $ trash put <path1> <path2> <path3>

List items in trash bin $ trash list

Restore or empty a file/files $ trash restore <file1> <file2> $ trash empty <file1> <file2> All files flag $ trash restore --all $ trash empty --all
