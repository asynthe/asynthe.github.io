  • Setting up pfSense

    Unzip the pfSense iso with gunzip like this gunzip pfSense iso .gz <filename>

    When making the VM, choose BSD as the type, and FreeBSD (64-bit) as the version.

    Set the memory size to 1024MB Select VDI (VirtualBox Disk Image) Choose dynamically allocated and set the size to 5GB

    • Setting up the pfSense Network

      pfSense will act as a gatekeeper between the internet and your internal network, making it secure from outside attackers. We are creating a safe place to add vulnerable machines and attack them.

      Click on Settings -> Network -> change NAT to Bridged Adapter, and choose your wireless/Ethernet card. It should be wlan0.

      This will create a direct connection between the pfSense virtual machine and the internet.

      Click on Adapter 2 -> change Attached to: to Internal Network (we will name it Internal LAN)

      —

      Run the pfSense VM, and proceed to install using the defaults. After installing, close the VM and proceed to remove the ISO from loading.
