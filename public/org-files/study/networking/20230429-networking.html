internet through Ethernet with nm-tui / network manager applet (2 computers)

So it has been usual that i have my laptop with internet and another machine that doesn't have a wifi card nor i have a wifi dongle to use, in this case, we can connect both devices using a ethernet cable like this.

In laptop or PC w internet

  1. Open nm-tui / network manager applet
  2. Inside the applet or TUI add a connection, choose Ethernet
  3. Edit the connection, go to IPv4 Settings
  4. Choose Method: Shared to other computers

Now, you may want to choose

ipconfig / ip a - NO CARRIER

This means that the connection is not being detected, check the cables

The "NO CARRIER" message indicates that the network interface is not receiving a carrier signal from the connected network. This can happen if the cable is disconnected, there is a hardware issue, or the network is down. It's a way for the system to inform you that there is no active connection on that particular network interface.

If you encounter the "NO CARRIER" message while using ipconfig, you should check the physical connection of the network cable and ensure that it is securely connected. If the issue persists, you may need to troubleshoot further, such as checking the network switch or contacting your network administrator.
