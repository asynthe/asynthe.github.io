Starship

main page github page

package

arch -> starship gentoo -> app-shells/starship nixos -> starship

install

after getting the package, add the next to your ~/.bashrc or ~/.zshrc

~/.bashrc 

eval "$(starship init bash)"

~/.zshrc 

eval "$(starship init zsh)"

config

all configuration is done in ~/.config/starship.toml or any other directory setting up env variable export STARSHIP_CONFIG=~/.config/starship/starship.toml

zsh-vi-mode

github page

clone the repo $ git clone https://github.com/jeffreytse/zsh-vi-mode.git ~/.config/zsh/plugins/zsh-vi-mode

source it in your .zshrc (or .bashrc) 

source $ZDOTDIR/plugins/zsh-vi-mode/zsh-vi-mode.plugin.zsh

zsh-autocomplete

github page

clone the repo $ git clone --depth 1 -- https://github.com/marlonrichert/zsh-autocomplete.git $ZDOTDIR/plugins/zsh-autocomplete

source it in your .zshrc (or .bashrc) 

source $ZDOTDIR/plugins/zsh-autocomplete/zsh-autocomplete.plugin.zsh
