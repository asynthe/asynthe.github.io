How to set up Fonts

on a user level copy and paste font into one of the next directories: ~/.fonts or ~/.local/share/fonts

on a system level copy and paste into following directory: /usr/share/fonts

now, make the new fonts cache with $ fc-cache (all fonts directories) or $ fc-cache -vf /usr/share/fonts (specific directory) -v -> verbose -f -> force
